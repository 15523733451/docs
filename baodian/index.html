    # 搭建指南

    # 环境搭建

    • 需要node环境和npm支持，如果不会安装npm请转到：安装指引

    # 安装初始化

    • 全局安装

      $ npm install -g vuepress
      1

    • 创建个文件夹作为目录

      $ mkdir vuepress-blog
      1

    • 项目初始化,初始化后会生成一个package.json文件

    • 在当前目录中创建一个docs目录

      $ cd vuepress-blog
$ npm init -y
$ mkdir docs

    # 核心配置

    • 在docs目录下创建.vuepress目录

      $ cd docs
$ mkdir .vuepress
# 主要存放配置
      1
      2
      3

    • 新建总配置文件config.js

      $ cd .vuepress
$ touch config.js
# config是整个项目的核心配置文件，所有菜单、栏目相关的配置均配置在该模块中
      1
      2
      3

    • 在config.js中加入内容

      module.exports = {
  title: '迦百农',
  description: '愿每一个孩子都被温柔以待',
  head: [
      ['link', {rel: 'icon', href: '/img/logo.jpg'}]
  ],
  markdown: {
      lineNumbers: true
  },
  themeConfig: {
      nav: require('./nav'),
      sidebar: 'auto',
      sidebar: {'/guide/':[
          {
                title:'新手指南',
                collapsable: true,
                children:[
                  '/guide/notes/one',
                ]
              },
              {
                title:'赞恩.阿尔.拉菲亚',
                collapsable: true,
                children:[
                  '/guide/notes/two',
                ]
              }
          ]
      },
      sidebarDepth: 2,
      lastUpdated: 'Last Updated',
      searchMaxSuggestoins: 10,
      serviceWorker: {
          updatePopup: {
              message: "有新的内容.",
              buttonText: '更新'
          }
      },
      editLinks: true,
      editLinkText: '在 GitHub 上编辑此页 ！'
  }
}
